# 拾物GlobePick 法律文档网站

这个文件夹包含用于 App Store 上架的隐私政策和服务条款网页。

## 📁 文件说明

- `privacy-policy.html` - 隐私政策网页
- `terms-of-service.html` - 服务条款网页

## 🚀 快速部署

### 方法1：GitHub Pages（推荐）

1. 在 GitHub 创建新仓库（公开）
2. 上传这两个 HTML 文件
3. 在仓库设置中启用 GitHub Pages
4. 获取链接：`https://你的用户名.github.io/仓库名/privacy-policy.html`

### 方法2：Vercel（推荐）

```bash
# 安装 Vercel CLI
npm install -g vercel

# 部署
cd website
vercel
```

### 方法3：Netlify

1. 访问 https://www.netlify.com
2. 拖拽 `website` 文件夹到 Netlify
3. 获取链接

详细步骤请查看：`部署隐私政策和服务条款-完整指南.md`

## 📝 使用说明

部署完成后，在 App Store Connect 中：
- **隐私政策 URL**：填入你的隐私政策链接
- **服务条款 URL**：填入你的服务条款链接（如需要）

## 🔗 链接示例

- 隐私政策：`https://你的域名/privacy-policy.html`
- 服务条款：`https://你的域名/terms-of-service.html`
